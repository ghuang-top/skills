**格式：** `<type>: <description>`

**类型定义：**
- feat: 新功能
- fix: 修复bug
- perf: 性能优化
- refactor: 重构代码
- docs: 文档更新
- style: 代码格式化（不影响功能）
- chore: 构建工具或依赖更新
- build: 构建系统修改
- test: 测试相关
- ci: CI/CD 配置
